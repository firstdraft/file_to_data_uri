== Setup

To install _with_ security, run:

[source,bash]
----
# ðŸ’¡ Skip this line if you already have the public certificate installed.
gem cert --add <(curl --compressed --location https://undefined.io/gems.pem)
gem install data_uri --trust-policy HighSecurity
----

To install _without_ security, run:

[source,bash]
----
gem install data_uri
----

You can also add the gem directly to your project:

[source,bash]
----
bundle add data_uri
----

Once the gem is installed, you only need to require it:

[source,ruby]
----
require "data_uri"
----

== Usage

== Development

To contribute, run:

[source,bash]
----
git clone https://github.com/undefined/data_uri
cd data_uri
bin/setup
----

You can also use the IRB console for direct access to all objects:

[source,bash]
----
bin/console
----

== Tests

To test, run:

[source,bash]
----
bin/rake
----
